# Humanised Jobnames

A small python library and command line wrapper to create humanised jobnames

## Example - command line

    ./bin/humanised-jobname.py 
    flaky-mogadishu

    ./bin/humanised-jobname.py 
    voiceless-damascus

## Example - flask

    FLASK_APP=bin/humanised-flask.py flask run

    curl http://127.0.0.1:5000/
    lively-taipei
